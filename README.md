# Maybac Cafe — Official Website

Live site → [maybac-srt.vercel.app](https://maybac-srt.vercel.app)

A fully responsive, single-page landing website I built for **Maybac Cafe** — a premium cafe in Vesu, Surat. Designed and developed from scratch to give the cafe a proper online presence with a clean, elegant look that matches their brand vibe.

---

## Pages / Sections

The site is a single-page layout with smooth scroll navigation across 5 sections:

**Home** — Full-screen hero with the cafe name, tagline ("Where Every Cup Tells a Story"), and two CTAs — one to the gallery, one to the contact section. Has a scroll indicator animation.

**About** — Tells the cafe's story, highlights four pillars (Premium Coffee, Gourmet Menu, Elegant Ambiance, 5-Star Service) with icons, and animated counters for happy customers, rating, and menu items.

**Experience** — A full-width feature section with a large image and copy about the cafe's atmosphere. More of a visual break than a content section.

**Gallery** — Photo grid of 10+ real images from inside the cafe — food, drinks, interiors, and ambiance shots. Clean masonry-style layout.

**Menu** — Displays 4 menu card images so visitors can browse before visiting.

**Contact / Find Us** — Location (with a Google Maps link), opening hours (Mon–Sun, 10 AM – 10 PM), Google review rating (4.5★ based on 67 reviews), and links to Instagram, Google, and Swiggy.

---

## Built with

- **HTML5, CSS3, Vanilla JS** — no frameworks, written from scratch
- Smooth scroll, scroll-triggered animations, animated number counters
- Fully **responsive** — works on mobile, tablet, and desktop
- Deployed on **Vercel**

## Features worth noting

- Sticky navbar that changes style on scroll
- Smooth section-based navigation with anchor links
- Animated stat counters (count up on scroll into view)
- Photo gallery with hover effects
- Google Maps deep link for directions
- Direct links to Instagram, Google Reviews, and Swiggy listing
- SEO meta tags included (description, keywords, viewport)
